# Tickr

Time management meets time tracking.


## Features

- Create, update and delete tasks with status tracking.
- Built-in timer for each task.
- Find tasks by status, priority or text search.
- End-to-end type safety with tRPC

## Stack

- Front-end | React, TypeScript, Tailwind CSS, Recharts
- Back-end | Node.js, Express, tRPC
- Validation | Zod
- Tools | Vite, TSX, Vitest
